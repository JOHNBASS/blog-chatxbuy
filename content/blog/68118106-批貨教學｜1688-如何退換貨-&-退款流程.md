<p><span style="-webkit-text-size-adjust:auto; -webkit-text-stroke-width:0px; caret-color:rgb(0, 0, 0); color:rgb(0, 0, 0); display:inline !important; float:none; font-family:-webkit-standard; font-size:medium; font-style:normal; font-variant-caps:normal; font-weight:400; letter-spacing:normal; orphans:auto; text-align:start; text-decoration:none; text-indent:0px; text-transform:none; white-space:normal; widows:auto; word-spacing:0px">網購商品因無法事先確認商品品質，有可能購買到不符合自己期望的商品品項，這種時候該怎麼辦呢？</span></p>

<p><span style="-webkit-text-size-adjust:auto; -webkit-text-stroke-width:0px; caret-color:rgb(0, 0, 0); color:rgb(0, 0, 0); display:inline !important; float:none; font-family:-webkit-standard; font-size:medium; font-style:normal; font-variant-caps:normal; font-weight:400; letter-spacing:normal; orphans:auto; text-align:start; text-decoration:none; text-indent:0px; text-transform:none; white-space:normal; widows:auto; word-spacing:0px">本篇文章將詳細介紹，若收到商品需要退換貨該如何處理，以及代買幫能提供什麼樣的協助哦～</span></p>

<hr>
<p><strong><span style="font-size:22px"><span style="-webkit-text-stroke-width:0px; caret-color:rgb(0, 0, 0); color:rgb(0, 0, 0); display:inline !important; float:none; font-family:-webkit-standard; font-style:normal; font-variant-caps:normal; font-weight:400; letter-spacing:normal; text-align:start; text-decoration:none; text-indent:0px; text-size-adjust:auto; text-transform:none; white-space:normal; word-spacing:0px">一、購買中國商品能不能退換貨或是退款呢？</span></span></strong></p>

<p><span style="-webkit-text-size-adjust:auto; -webkit-text-stroke-width:0px; caret-color:rgb(0, 0, 0); color:rgb(0, 0, 0); display:inline !important; float:none; font-family:-webkit-standard; font-size:medium; font-style:normal; font-variant-caps:normal; font-weight:400; letter-spacing:normal; orphans:auto; text-align:start; text-decoration:none; text-indent:0px; text-transform:none; white-space:normal; widows:auto; word-spacing:0px">購買1688上的中國商品，若遇到商品瑕疵或品項不符等，只要可以「</span><span style="font-size:16px"><strong style="-webkit-text-size-adjust:auto; -webkit-text-stroke-width:0px; caret-color:rgb(0, 0, 0); color:rgb(0, 0, 0); font-style:normal; font-variant-caps:normal; letter-spacing:normal; orphans:auto; text-align:start; text-decoration:none; text-indent:0px; text-transform:none; white-space:normal; widows:auto; word-spacing:0px"><span style="--original-color:#996900; color:rgb(153, 105, 0)" textcolor="#996900">明顯歸咎於廠商的狀況</span></strong></span><span style="-webkit-text-size-adjust:auto; -webkit-text-stroke-width:0px; caret-color:rgb(0, 0, 0); color:rgb(0, 0, 0); display:inline !important; float:none; font-family:-webkit-standard; font-size:medium; font-style:normal; font-variant-caps:normal; font-weight:400; letter-spacing:normal; orphans:auto; text-align:start; text-decoration:none; text-indent:0px; text-transform:none; white-space:normal; widows:auto; word-spacing:0px">」，基本上都是可以申請退換貨或是退款的，但若是運送過程中產生的損壞等，難以證明為廠商疏失的部分則很難成功申請，進口商品因物流時間長且中間經過多次物流轉手，商品難免會有較多碰撞的發生，所以推薦大家盡量購買承受碰撞能力較高的商品哦，如：玻璃罐、電子儀器等較不耐碰撞的商品就不推薦購買。</span><br style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0); font-style: normal; font-variant-caps: normal; font-weight: 400; letter-spacing: normal; orphans: auto; text-align: start; text-indent: 0px; text-transform: none; white-space: normal; widows: auto; word-spacing: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px; text-decoration: none;">
<br style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0); font-style: normal; font-variant-caps: normal; font-weight: 400; letter-spacing: normal; orphans: auto; text-align: start; text-indent: 0px; text-transform: none; white-space: normal; widows: auto; word-spacing: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px; text-decoration: none;">
<span style="-webkit-text-size-adjust:auto; -webkit-text-stroke-width:0px; caret-color:rgb(0, 0, 0); color:rgb(0, 0, 0); display:inline !important; float:none; font-family:-webkit-standard; font-size:medium; font-style:normal; font-variant-caps:normal; font-weight:400; letter-spacing:normal; orphans:auto; text-align:start; text-decoration:none; text-indent:0px; text-transform:none; white-space:normal; widows:auto; word-spacing:0px">還有需注意的是，</span><span style="font-size:16px"><span bgcolor="#ffe600" style="-webkit-text-size-adjust:auto; -webkit-text-stroke-width:0px; background-color:rgb(255, 230, 0); caret-color:rgb(0, 0, 0); color:rgb(0, 0, 0); font-style:normal; font-variant-caps:normal; font-weight:400; letter-spacing:normal; orphans:auto; text-align:start; text-decoration:none; text-indent:0px; text-transform:none; white-space:normal; widows:auto; word-spacing:0px">進口中國商品較難使用鑑賞期為由申請退款</span></span><span style="-webkit-text-size-adjust:auto; -webkit-text-stroke-width:0px; caret-color:rgb(0, 0, 0); color:rgb(0, 0, 0); display:inline !important; float:none; font-family:-webkit-standard; font-size:medium; font-style:normal; font-variant-caps:normal; font-weight:400; letter-spacing:normal; orphans:auto; text-align:start; text-decoration:none; text-indent:0px; text-transform:none; white-space:normal; widows:auto; word-spacing:0px">，除多數商家本身就未提供鑑賞期退款外，正常狀況下阿里巴巴系統是以交付至代買幫倉庫開始計算完成交易的時間，因跨國運輸商品時間較長，買家收到貨後很容易超過訂單服務時間。</span><br style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0); font-style: normal; font-variant-caps: normal; font-weight: 400; letter-spacing: normal; orphans: auto; text-align: start; text-indent: 0px; text-transform: none; white-space: normal; widows: auto; word-spacing: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px; text-decoration: none;">
<br style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0); font-style: normal; font-variant-caps: normal; font-weight: 400; letter-spacing: normal; orphans: auto; text-align: start; text-indent: 0px; text-transform: none; white-space: normal; widows: auto; word-spacing: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px; text-decoration: none;">
<span style="-webkit-text-size-adjust:auto; -webkit-text-stroke-width:0px; caret-color:rgb(0, 0, 0); color:rgb(0, 0, 0); display:inline !important; float:none; font-family:-webkit-standard; font-size:medium; font-style:normal; font-variant-caps:normal; font-weight:400; letter-spacing:normal; orphans:auto; text-align:start; text-decoration:none; text-indent:0px; text-transform:none; white-space:normal; widows:auto; word-spacing:0px">另外，若收到商品不滿意外型、顏色等非產品本身缺陷的原因需要退款，廠商多數也不願意同意，所以在大量進貨前建議大家先進幾個少量樣品，看看商品的品質是不是符合自身期待哦～</span><br style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0); font-style: normal; font-variant-caps: normal; font-weight: 400; letter-spacing: normal; orphans: auto; text-align: start; text-indent: 0px; text-transform: none; white-space: normal; widows: auto; word-spacing: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px; text-decoration: none;">

<p><span style="-webkit-text-size-adjust:auto; -webkit-text-stroke-width:0px; caret-color:rgb(0, 0, 0); color:rgb(0, 0, 0); display:inline !important; float:none; font-family:-webkit-standard; font-size:medium; font-style:normal; font-variant-caps:normal; font-weight:400; letter-spacing:normal; orphans:auto; text-align:start; text-decoration:none; text-indent:0px; text-transform:none; white-space:normal; widows:auto; word-spacing:0px">商品若確定需要申請退款，可以依照下列流程處理：</span><br style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0); font-style: normal; font-variant-caps: normal; font-weight: 400; letter-spacing: normal; orphans: auto; text-align: start; text-indent: 0px; text-transform: none; white-space: normal; widows: auto; word-spacing: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px; text-decoration: none;">
<br style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0); font-style: normal; font-variant-caps: normal; font-weight: 400; letter-spacing: normal; orphans: auto; text-align: start; text-indent: 0px; text-transform: none; white-space: normal; widows: auto; word-spacing: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px; text-decoration: none;">
<span style="font-size:16px"><strong style="-webkit-text-size-adjust:auto; -webkit-text-stroke-width:0px; caret-color:rgb(0, 0, 0); color:rgb(0, 0, 0); font-style:normal; font-variant-caps:normal; letter-spacing:normal; orphans:auto; text-align:start; text-decoration:none; text-indent:0px; text-transform:none; white-space:normal; widows:auto; word-spacing:0px">Step1：提供商品圖及開箱錄影給代買幫客服，並詳述商品狀況及退貨原因</strong><br style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0); font-style: normal; font-variant-caps: normal; font-weight: 400; letter-spacing: normal; orphans: auto; text-align: start; text-indent: 0px; text-transform: none; white-space: normal; widows: auto; word-spacing: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px; text-decoration: none;">
<span style="-webkit-text-stroke-width:0px; caret-color:rgb(0, 0, 0); color:rgb(0, 0, 0); display:inline !important; float:none; font-family:-webkit-standard; font-style:normal; font-variant-caps:normal; font-weight:400; letter-spacing:normal; text-align:start; text-decoration:none; text-indent:0px; text-size-adjust:auto; text-transform:none; white-space:normal; word-spacing:0px">代買幫客服收到以上資訊後，將會反應給出貨廠商，並協調能否申請退貨、退款，廠商確認能夠退款後，就會提供客戶退貨地址，並通知買家進行下一個步驟。</span><br style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0); font-style: normal; font-variant-caps: normal; font-weight: 400; letter-spacing: normal; orphans: auto; text-align: start; text-indent: 0px; text-transform: none; white-space: normal; widows: auto; word-spacing: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px; text-decoration: none;">
<br style="caret-color: rgb(0, 0, 0); color: rgb(0, 0, 0); font-style: normal; font-variant-caps: normal; font-weight: 400; letter-spacing: normal; orphans: auto; text-align: start; text-indent: 0px; text-transform: none; white-space: normal; widows: auto; word-spacing: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px; text-decoration: none;">
